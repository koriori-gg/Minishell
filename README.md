## 🛠️ Usage
### Requirements
The function is written in C language and thus needs the gcc compiler and some standard C libraries to run.

### Instructions
#### 1.clone
```
git clone git@github.com:koriori-gg/Minishell.git
```
#### 2. Using it in your code
```
cd Minishell
make
./minishell
```
